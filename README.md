# 🕹️ Simon Says: Rickroll Edition (RPi Pico)

A fun, interactive memory game built with MicroPython on the Raspberry Pi Pico. Repeat the sequence of lights and sounds to progress. If you fail, the buzzer will "Rickroll" you by playing the chorus of *Never Gonna Give You Up*!

## 🛠️ Hardware Requirements

| Component | Quantity | Description |
| :--- | :--- | :--- |
| **Raspberry Pi Pico** | 1 | Microcontroller |
| **LEDs** | 3 | Any colors (Red, Green, Blue recommended) |
| **Resistors** | 3 | 220Ω resistors for the LEDs |
| **Push Buttons** | 3 | Tactile switches |
| **Buzzer** | 1 | Piezo buzzer (with a ring/housing) |
| **Jumper Wires** | - | M-M or M-F cables |

---


## 🎮 How to Play

1. **Watch:** The game will flash a random LED and play a tone.
2. **Repeat:** Press the corresponding button.
3. **Advance:** If you are correct, the game adds one more light to the sequence.
4. **Lose:** If you press the wrong button, all LEDs light up and the buzzer plays the Rickroll melody!
